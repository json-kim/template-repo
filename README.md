# template-repo
템플릿으로 사용될 레포입니다

- Label
- Pull Request 템플릿
- Issue 템플릿
- Release 템플릿

## 라벨 적용
```
npx github-label-sync --access-token {access_token} --labels ./labels.json {repository}
```
